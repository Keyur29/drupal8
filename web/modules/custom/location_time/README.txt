Module is regarding to see current location and time and will work on Drupal 8 and 9 versions.

CONFIGURATION
--------------------------------

1) Go to admin->Configuration->System->Location time configuration
2) To get timezone options dynamically, add the values with comma separated in "Timezone Options" field and save the form
Eg : America/Chicago,America/New_York,Asia/Tokyo,Asia/Dubai,Asia/Kolkata,Europe/Amsterdam,Europe/Oslo,Europe/London
3) Enter country, city and select timezone and save the form
